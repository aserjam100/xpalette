# xpalette

A color palette maker built in C with Raylib, RayGui, and raymath.

This is a learning project — I'm using it to get comfortable with C and the Raylib ecosystem.

## Features (planned)

- Interactive color picker (HSV)
- Build and edit palettes of up to N colors
- Generate harmonies (complementary, analogous, triadic, etc.)
- Export to common formats (hex list, GIMP `.gpl`, CSS)
